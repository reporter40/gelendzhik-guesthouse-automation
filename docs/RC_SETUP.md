# Настройка RealtyCalendar

## Обязательно
1. Подтвердить email в RC (dm.martyshenko@gmail.com)
2. Настроить сезоны: RC → Настройки → Сезоны
3. Проверить AmoCRM интеграцию

## REDATA (динамические цены)
Написать @redata_support_bot в Telegram с текстом: "Хочу подключить REDATA. RC #79366, 4 объекта в Геленджике."

## Автопилот (138₽/день)
RC → Настройки → Интеграции → Автопилот. Включать только в сезон (май-октябрь).

## Webhooks
Если доступны — URL: https://n8n.domain.ru/webhook/rc-booking. Если нет — использовать polling (workflow 01_ical_sync).
