## Критерии отбора для №50

Объект №50 — это дом/этаж под ключ для большой семьи:
- Вместимость: **6–8 гостей**
- Тип: дом, этаж под ключ, крупные апартаменты, коттедж, таунхаус
- Площадь: **50–85 м²**
- Обязательно: **своя кухня**
- Желательно: **двор, терраса, зона отдыха, отдельный вход**
- Расположение: Геленджик, Толстый мыс / центр / не далее 3–5 км от моря
- Цена: **6 500–11 000 ₽/ночь** (сезон 2026)
- Формат: семейный, min stay 3–7 дней

## Шкала similarity_score для №50

| Score | Критерии |
|-------|----------|
| **90–100** | 6–8 гостей, дом/этаж/крупные апарт., своя кухня, двор/терраса/территория, близко к морю (<1 км), цена 6.5–11К, семейный формат |
| **75–89** | Большинство критериев, но нет одного важного (нет двора ИЛИ чуть дальше от моря ИЛИ 5–6 гостей) |
| **60–74** | Частичное совпадение: например, 6 гостей но квартира без двора, или дом но 4–5 гостей |
| **< 60** | Не подходит: rejected / background only |

---

## signal_quality_score

| Score | Когда |
|-------|-------|
| **0.8–1.0** | Актуальная цена 2026, прямая ссылка работает, много параметров из листинга |
| **0.6–0.79** | Цена из поиска 2026, часть параметров оценочная |
| **0.4–0.59** | Цена примерная, некоторые параметры не известны |
| **< 0.4** | Цена устаревшая или из description без прямой верификации |
